# Review Skill

指定されたPR番号のプルリクエストをgit-reviewエージェントを使用して包括的にレビューしてください。

ポイントは以下です。

- GitHub、Linear、Notion MCPを使用した多角的な情報収集
- PRのpullとローカル環境での詳細なコード分析
- 外部ドキュメントとコード実装の整合性確認
- セキュリティ、パフォーマンス、品質の包括的評価
